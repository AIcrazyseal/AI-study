## 二、Agent学习资源
1.[GitHub上6个Agent项目简介](https://zhuanlan.zhihu.com/p/1986406466626810036)
* [hello-agents：国内社区Datawhale开源的教程+仓库代码](https://github.com/datawhalechina/hello-agents)，教你使用Coze、Dify、n8n等平台快速搭建应用，适合快速验证想法。
* [500-AI-Agents-Projects：超过500个AI Agent落地案例](https://github.com/ashishpatel26/500-AI-Agents-Projects)，按医疗、金融、教育、DevOps 等垂直、细分领域对Agent项目进行了详细分类，包括CrewAI、Autogen、Agno、Langgraph等主流框架的实际应用。
* [GenAI_Agents：技术博主NirDiamant打造>40种场景的案例](https://github.com/NirDiamant/GenAI_Agents)，包含了大量可直接运行的Jupyter Notebook教程，覆盖了从入门到精通的全方位场景，集成了LangChain、LangGraph、AutoGen 等主流框架，以及 MCP 等先进技术。
* [agents-course核心理念：编码Agents](https://github.com/huggingface/agents-course)，让LLM直接写Python代码来解决问题，这门课所有的练习都可以在 Hugging Face Spaces 上直接运行。你甚至不需要自己在本地配环境，浏览器点开就能跑，还能直接用 HF 提供的免费 Inference API。
* [微软ai-agents-for-beginners：10节课程，讲解企业级开发中验证过的模式和框架](https://github.com/microsoft/ai-agents-for-beginners)，微软主推SDK Semantic Kernel，专注于将大模型集成到现有代码中，是企业级应用的首选。还涉及应用处理复杂Multi-Agent协作的开源框架AutoGen。
* [6周学会AI智能体:全框架覆盖](https://github.com/ed-donner/agents),横向对比并实战了OpenAI Agents SDK、CrewAI、LangGraph、AutoGen四大主流框架，还率先涵盖了最新的 MCP。
