# GitHub等相关知识学习

## 一、代码托管平台

1.GitHub：国际最大的代码托管平台[官方文档](https://docs.github.com/zh)

2.Gitee：中国最大的代码托管平台[官方文档](https://gitee.com/)

3.GitHub Desktop：Git的图形化代码托管应用程序[官方文档](https://docs.github.com/zh/desktop)

4.Git：GitHub的版本控制系统

* [官方文档](https://git-scm.com/)
* [Git 命令的完整参考指南](https://git-scm.com/docs)

5.[国外GitHub与中国Gitee联系与区别:](https://zhuanlan.zhihu.com/p/1905942793676915767)

* 相同点：
  * 都基于 Git分布式版本控制系统，核心命令（如 git clone, git push, git pull）完全一致。
  * 均支持代码托管、分支管理、Pull Request（Merge Request）、Issues 等基础功能。
* 不同点：
  * Gitee 是国内的托管平台，针对中文用户优化，服务器位于中国大陆，访问速度更快。
  * GitHub/GitLab 是国际平台，服务器主要在海外，功能更国际化（如全球开发者社区、CI/CD 工具链更成熟）。
